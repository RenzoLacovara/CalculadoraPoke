# Calculadora de Efectividad
## Sobre el Proyecto
Calculadora que resuelve eficiencia y resistencia según el tipo de Pokémon usando JavaScript y PokeAPi, cuenta con responsive design para poder utilizarla en cualquier dispositivo y alertas personalizadas que indican cuantos pokemon comparten el tipo seleccionado, actualizada dia a dia, se trata de mi proyecto final para el Curso de JavaScript de CoderHouse
